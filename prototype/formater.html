Formater : Formater(N.format)는 입력한 값들을 지정한 포멧 룰대로 양식화(formatting)한 값을 반환하거나 입력요소에 표시 해 줍니다.

[Format(Command)] 버튼을 클릭하면 Data 영역에 있는 JSON 데이터를 지정한룰(소스 참고)대로 포멧팅하여 Command Result 부분에 표시 해 줍니다.

[Format] 버튼을 클릭하면 데모영역에 있는 HTML 요소들에 각 요소들의 class 속성에 정의된 포멧룰을 기반으로 데이터를 양식화 하여 해당요소에 표시 해 줍니다.

입력요소에 Formater 가 바인드 되면 format 과 unformat 이벤트를 생성합니다.

[Run unformat event] 버튼을 클릭하면 Formater 가 적용된 입력요소의 format 이벤트를 실행 합니다. format 이벤트는 해당입력요소에 입력된 값을 양식화 해 줍니다.

[Run format event] 버튼을 클릭하면 Formater 가 적용된 입력요소의 unformat 이벤트를 실행 합니다. unformat 이벤트는 해당입력요소에 입력된 값을 원래 바인드 되어 있는 값으로 돌려 줍니다.

class='{ "format" : [["trimtoempty"], ["limit", "13", "..."]] }'
class='{ "format" : [["date", 12]] }'
