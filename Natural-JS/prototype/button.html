Button : Button(N.button)은 a , input button , button 요소들을 옵션으로 지정한 모양으로 버튼을 만들어 줍니다.

버튼의 옵션은 해당 요소의 class attribute에 JSON 형태의 텍스트로 선언하여 지정 할 수 있고 버튼을 초기화 할 때 직접 JSON Object 형태로 지정 할 수 있습니다.

[Make Button] 버튼을 클릭하면 지정한 옵션이 반영된 버튼을 볼 수 있습니다.

DEMO

Default options :

size color
disable If this checkbox with check(true), buttons are initailized with disabled effect If this checkbox with uncheck(false), when buttons are enabled or disabled, the effect does not apply
