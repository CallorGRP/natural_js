공공아이핀

1. 사전 준비 사항
 - 공공아이핀 공무원 창구 : http://www.g-pin.go.kr:8080
   * 문의는 해당사이트에 콜센터 전화번호가 있음.

 - 개인용 행정전자서명인증서(개인용 GPKI)
   * 공무원 창구 로그인 할 때 필요
   * 대부분 공무원들은 가지고 있고 없다면 http://www.gpki.go.kr 접속 후 메인화면 중앙에 행정전자서명 신청서 클릭 후 안내에 따라 개인용으로 신청

 - 개인용 행정전자서명인증서(서버용 GPKI)
   * 공공아이핀 신청할때, 모듈 설치 할 때 필요
   * 서버인증서 신청 http://www.gpki.go.kr 접속 후 메인화면 중앙에 행정전자서명 신청서 클릭 후 안내에 따라 신청
   * 서버인증서 저장 위치 : C:\GPKI\certificate\class1 <-- 요 폴더의 파일들과 인증서 패스워드를 따로 받아놔 주세요.

 - 방화벽 오픈 : 
   * 클라이언트 방화벽 오픈 : 클라이언트 PC 가 모든 포트가 열려있으면 상관 없지만 내부 망으로 운영 된다면 아래 주소와 포트에 대한 인터넷 접속이 가능 해야 함.
     + 문서가 집에 있어서 집에 가서 확인 후 보내 드릴께요
   * 서버 방화벽 오픈 : 아래 호스트와 포트를 열어줘야 함.
     + 문서가 집에 있어서 집에 가서 확인 후 보내 드릴께요


2. 공공 I-PIN 서비스 연계를 위한 행정처리(공무원)

   2.1. 공공 I-PIN 서비스 연계 신청을 위한 공문처리
     2.1.1. 공무원 창구 접속 후 담당자 GPKI 인증서로 로그인
     2.1.2. 서비스이용신청서를 다운받아 이메일로 서비스 연계신청.
     2.1.3. 담당자 지정 신청 화면에서 입력 항목들을 입력 후 해당업무 단락에서 이용기관 담당자 등록 선택 후 신청
     2.1.4. 승인날때 까지 기다립니다. 17시 이후 신청접수(익일 오전 12시 이전). 17시 이전 신청접수(18시)
     2.1.5. 공공아이핀 공무원 창구로 접속해서 승인이 났으면 이용기관 사이트 신청 탭으로 가서 정보들을 입력 후 신청을 합니다.
        * 가입자 정보 동기화 페이지 주소와 인증정보 수신 페이지 주소는 도메인만 변경하고 도메인 뒤 주소들은 그대로 나둬 주세요.
          도메인은 개발서버의 IP로 지정 해 주시고 개발서버가 없다면 제가 개발할 PC의 아이피로 지정 해 주세요. 이미 오픈한 상태라면 운영서버의 도메인으로 지정 해 주세요. 
          위 테스트 장비들은 사전준비사항의 방화벽 오픈 관련 주소와 포트들이 열려 있어야 합니다.
        * 서버인증서 등록은 위 사전 준비 사항의 서버인증서를 지정 해 주세요(C:\GPKI\certificate\class1).
     2.1.5. 신청이 완료 되면 담당사이트 관리 탭으로 가면 SITE-ID 를 볼 수 있습니다. 이걸 따로 적어두세요. 매우중요한 값 입니다.

   2.2. 위 절차가 이해하기 어려우면 공공아이핀 센터에 전화 해서 알려주는대로 처리 하면 됩니다.

   2.3. 모두 등록이 완료 되었으면 연계가이드(www.g-pin.go.kr:8080/G-PIN_Client) 으로 접속해서 연계 모듈을 다운로드 받아놔 주세요.
        * 연계모듈은 JDK 1.5 이상에서 구동 됩니다.

   2.4. 테스트서버나 운영서버에서 작업 해야 된다면 서버에 G-PIN 이라는 컨텍스트를 잡고 해당 모듈을 올려 주세요.


3. 행정 처리 요약
   1. GPKI 서버인증서 발급 후 따로 저장해 둠
   2. 공공아이핀 사용신청서 발송 하고 담당자지정 신청 후 승인을 기다림
   3. 공공아이핀 연계 정보 등록 후 SITE-ID를 따로 적어둠.
   4. 연계 모듈 다운로드 후 서버에 올려 놓고 받을 모듈 파일을 따로 저장 해 둠.
