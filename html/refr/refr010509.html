Pagination(N.pagination)

개요

Pagination(이하 N.pagination)은 데이터 표현에 대한 브라우저의 부하를 줄이기 위해 데이터를 지정한 갯수만큼 잘라서 보여주는 컴포넌트입니다. N.grid 와 연동하여 사용하면 서버단 페이징(DB 쿼리 페이징) 및 브라우저 단 페이징(전체 조회 후 잘라서 보여주기) 을 쉽게 구현 할 수 있습니다. 또한 페이징이 필요한 어떠한 목록도 페이징 처리를 할 수 있습니다.

API DEMO(문서 준비 중입니다.)

HTML 템플릿(HTML templete)

다음과 같은 구조의 요소들을 배치하면 Pagination이 만들어 집니다.

<div id="pagination">
	<ul>
		<li><a href="#" title="첫페이지">first</a></li>
		<li><a href="#" title="이전">prev</a></li>
	</ul>
	<ul>
		<li><a href="#"><span>1</span></a></li>
	</ul>
	<ul>
		<li><a href="#" title="다음페이지">next</a></li>
		<li><a href="#" title="마지막페이지">last</a></li>
	</ul>
</div>

생성자(Constructor)

Name Arguments Type Return Description
N.pagination - - N.pagination N.pagination의 객체 인스턴스를 생성합니다.
data JSON Array - 기본옵션의 data 옵션과 같습니다.
opts JSON Object - 기본 옵션을 JSON Object 형태로 지정 합니다.
N(obj).pagination - - N.pagination

N.pagination의 객체 인스턴스를 jQuery 플러그인 방식으로 생성합니다.

N.pagination과 동작되는 로직은 같으며 내부적으로 N()의 첫번째 인자가 N.pagination()의 첫 번째 인자로 지정 됩니다.
opts JSON Object - N.pagination의 opts 옵션과 같습니다.

기본옵션(Default options)

Name Type Default Required Description
data JSON Array undefined ○ context(HTML Templete)요소에 바인드 할 데이터
context jQuery Object obj ○ Pagination 을 적용할 컨텍스트 요소를 지정합니다.
totalCount int 0 ○ 페이징 될 전체 로우 카운트를 지정 합니다.
countPerPage int 10 ○ 한번에 몇개의 로우데이터를 반환 할 것인지 지정 합니다.
countPerPageSet int 10 ○ 페이징 번호를 몇개를 표시할 것인지 지정 합니다.
pageNo int 1 ○ 초기화시 처음 표시 할 페이지 번호를 지정 합니다.
onChange Function null 페이지가 전환 될때 발생되는 이벤트 입니다. 
onChange : function(pageNo, selEle, selData, currPageNavInfo) {
	// this 는 선택한 페이지 링크 요소
	// pageNo 는 선택한 페이지 번호
	// selEle 는 선택한 페이지 링크 요소
	// selData 는 선택한 페이지가 가지고 있는 데이터 리스트
	// currPageNavInfo 는 이벤트가 발생할 시점의 페이징 정보
}

함수(Methods)(문서 준비 중입니다.)

Name Arguments Type Return Description
data - - JSON Array | jQuery Object 컴포넌트에 바인드 되어 있는 최종 데이터를 반환 합니다.
selFlag boolean - 반환되는 데이터의 형태를 결정하는 구분자 입니다.

selFlag 인자 값에 따라 다음과 같은 형식의 데이터를 반환 합니다.

  • 입력하지 않았거나 undefined : JSON Array 형태의 데이터를 반환 합니다.
  • false : jQuery Object 형태의 데이터를 반환 합니다.
context - - jQuery Object 컴포넌트를 적용한 요소를 반환 합니다.
selector String - selector 문자열을 입력하면 해당 context 하위 요소에서 선택된 요소를 반환 합니다.
bind - - N.select 지정된 context에 데이터를 바인드 합니다.
data String - data인자를 지정하면 컴포넌트에 바인드 된 데이터가 지정한 데이터로 치환되어 바인드 됩니다.
totalCount int - 페이징 될 전체 로우 카운트가 다시 지정 됩니다.
