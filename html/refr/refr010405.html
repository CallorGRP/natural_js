Natural-DATA Libraries

    개요 Overview

    Natural-DATA 패키지에는 array[json object] 타입의 데이터를 조작할 수 있는 기능을 제공 합니다.

    함수 Methods

    Name Arguments Type Return Description
    N.data.refine - - 정제된 array[json object] 서버나 JSON 데이터가 담긴 파일에서 가져온 JSON 데이터를 Natural-JS에서 사용할수 있는 타입의 json 타입으로 만들어 줍니다.
    obj array[json object] - 정제 할 데이터
    listId string - 서버나 JSON 데이터가 담긴 파일에서 여러개의 리스트를 반환했을경우의 해당 리스트의 ID 값
    N.data.filter - - 필터링된 array[json object] 지정한 조건으로 array[json object] 에서 또다른 array[json object] 데이터를 추출 합니다.
    arr array[json object] 필터링 할 데이터
    callback function -

    필터링 조건을 입력 합니다.

    조건은 다음과 같이 callback function의 return 구문 다음에 필터링 조건을 명시(return 이 true 일 경우의 row data set 반환)합니다. 

    var fData = N.data.filter([
	{ name : "홍길동" },
	{ name : "Mike" }
], function(data) {
    return data.name === "Mike";
});

/*
fData의 값(결과값) :
[
	{ name : "Mike" }
]
*/
    N.data.sort - - 정렬된 array[json object]
    array[json object] 를 지정한 특정 key 명으로 정렬 합니다. arr array[json object] - 정렬 할 데이터
    key string - 정렬 기준이 될 key(컬럼) 명
    reverse boolean - 오름차순(true), 내림차순(false) 선택
    N(obj).datarefine - - 정제된 array[json object]

    N.data.refine을 플러그인 방식으로 초기화 하는 방식입니다.

    N.data.refine과 동작되는 로직은 같으며 실제 N() 의 인자로 들어온 값이 N.data.refine()의 첫 번째 인자로 지정 됩니다.
    listId string - N.data.refine의 listId 인자 와 같습니다.
    N(obj).datafilter - - 필터링된 array[json object]

    N.data.filter를 플러그인 방식으로 초기화 하는 방식입니다.

    N.data.filter와 동작되는 로직은 같으며 실제 N() 의 인자로 들어온 값이 N.data.filter()의 첫 번째 인자로 지정 됩니다.
    queryString array[json object] N.data.filter의 queryString 인자 와 같습니다.
    N(obj).datasort - - 정렬된 array[json object]

    N.data.sort를 플러그인 방식으로 초기화 하는 방식입니다.

    N.data.sort와 동작되는 로직은 같으며 실제 N() 의 인자로 들어온 값이 N.data.sort()의 첫 번째 인자로 지정 됩니다.
    key string - N.data.sort의 key 인자 와 같습니다.
    reverse boolean - N.data.sort의 reverse 인자 와 같습니다.
