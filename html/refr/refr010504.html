Popup(N.popup)

개요

Popup(이하 N.popup) 은 지정한 요소나 다른페이지를 레이어팝업 형태로 만들어 주는 컴포넌트입니다.

url 옵션으로 지정한 페이지를 가져올때 가져온 페이지의 Controller 에는 caller(호출한 팝업의 instance(N.popup)) 와 opener(팝업을 호출한 페이지(부모페이지)의 Controller) 속성이 추가 되어 부모 페이지와 자신을 호출한 팝업 객체를 직접 제어할 수 있습니다.

opener 객체는 팝업을 생성하는 구문의 위치에 따라 생성이 안 될 수 있습니다.

API DEMO

생성자(Constructor)

Name Arguments Type Return Description
N.popup - - N.popup N.popup의 객체 인스턴스를 생성합니다.
context window or jQuery Object - 기본옵션의 context 옵션과 같습니다.
opts JSON Object - 기본 옵션을 JSON Object 형태로 지정 합니다.
N(obj).popup - - N.popup

N.popup의 객체 인스턴스를 jQuery 플러그인 방식으로 생성합니다.

N.popup과 동작되는 로직은 같으며 내부적으로 N()의 첫번째 인자가 N.popup()의 첫 번째 인자로 지정 됩니다.
opts JSON Object - N.popup의 opts 옵션과 같습니다.

기본옵션(Default options)

Name Type Default Required Description
context jQuery Object obj 같은 페이지 내에 있는 블록(Block) 요소를 지정 합니다.
url
title String undefined

팝업의 제목,

title 옵션이 undefined 이면 타이틀바를 표시하지 않고 context 값으로 지정한 요소의 title 속성에 값이 있다면 해당 값으로 지정 됩니다.

button boolean true 버튼 표시 유무
modal boolean true 모달 적용 유무
height number 0 팝업의 높이
width number 0 팝업의 넓이
alwaysOnTop boolean false 팝업을 항상 페이지의 최상위로 항상 유지하기 위한 옵션
dynPos boolean true 브라우저 창 크기 조절 시 블록 오버레이의 크기도 같이 조절 할지 여부
draggable boolean false 메시지 창 드래그 가능 유무
onOk Function null 확인버튼을 눌렀을때 발생되는 이벤트
onCancel Function null 취소버튼을 눌렀을때 발생되는 이벤트
onOpen String null

팝업이 열릴 때마다 발생되는 이벤트

이벤트 콜백 메서드는 팝업페이지의 Controller 에 구현되어야 합니다. 메서드면은 Controller 에 정의한 메서드명칭만 String 으로 지정 합니다.

이벤트 콜백의 메서드는 인자로 onOpenData 가 들어옵니다.

url을 지정하여 다른 페이지를 가져왔을때만 사용가능 합니다.
onOpenData Object null

onOpen 이벤트의 인자로 들어가 팝업에 전달될 데이터

url을 지정하여 다른 페이지를 가져왔을때만 사용가능 합니다.

open 함수의 첫번째 argument 로 넣어줘도 됩니다(동적인 데이터 전달 시 사용).
onClose Function null

팝업이 닫힐 때마다 발생되는 이벤트.

이벤트 콜백의 메서드는 인자로 onCloseData 가 들어옵니다.

팝업 페이지에서 자신을 닫을때는 Controller에서 다음과 같이 처리 합니다.

this.caller.close(onCloseData)

url을 지정하여 다른 페이지를 가져왔을때만 사용가능 합니다.
onCloseData Object null

onClose 이벤트의 인자로 들어가 팝업의 부모페이지에 전달될 데이터

url을 지정하여 다른 페이지를 가져왔을때만 사용가능 합니다.

close 함수의 첫번째 argument 로 넣어줘도 됩니다(동적인 데이터 전달 시 사용).

함수(Methods)

Name Arguments Type Return Description
context - - jQuery Object 컴포넌트를 적용한 요소를 반환 합니다.
selector String - selector 문자열을 입력하면 해당 context 하위 요소에서 선택된 요소를 반환 합니다.
open onOpenData Object N.popup 팝업을 보여 줍니다.
close onCloseData Object N.popup 팝업을 숨깁니다.
remove - - N.popup 팝업을 표현하는 HTML Element들을 제거합니다.
