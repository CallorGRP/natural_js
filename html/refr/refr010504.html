Popup(N.popup)

개요

Overview

Popup(이하 N.popup) 은 지정한 요소나 다른페이지를 레이어팝업 형태로 만들어 주는 컴포넌트입니다.

url 옵션으로 지정한 페이지를 가져올때 가져온 페이지의 Controller 에는 caller(호출한 팝업의 instance(N.popup)) 와 opener(팝업을 호출한 페이지(부모페이지)의 Controller) 속성이 추가 되어 부모 페이지와 자신을 호출한 팝업 객체를 직접 제어할 수 있습니다.

opener 객체는 팝업을 생성하는 구문의 위치에 따라 생성이 안 될 수 있습니다.

생성자

Constructor

N.popup - - N.popup N.popup의 객체 인스턴스를 생성합니다.

N.popup의 객체 인스턴스를 jQuery 플러그인 방식으로 생성합니다.

N.popup과 동작되는 로직은 같으며 내부적으로 N()의 첫번째 인자가 N.popup()의 첫 번째 인자로 지정 됩니다.
opts JSON Object - N.popup의 opts 옵션과 같습니다.

팝업의 제목

title 옵션이 undefined 이면 타이틀바를 표시하지 않고 context로 지정한 요소의 title 속성에 값이 있으면 해당 값으로 지정 됩니다.
button boolean true 버튼 표시 유무
modal boolean true 모달 적용 유무
height number 0 팝업의 높이
width number 0 팝업의 넓이
alwaysOnTop boolean false 팝업을 항상 페이지의 최상위로 항상 유지하기 위한 옵션
dynPos boolean true 브라우저 창 크기 조절 시, 또는 부모 컨텐츠 높이가 동적으로 변경되었을때 블록 오버레이의 크기와 팝업 창의 위치를 자동으로 조절 할지 여부
draggable boolean false 팝업 창 드래그 가능 유무
draggableOverflowCorrection boolean true 메시지 창 드래그 시 팝업 창이 화면바깥으로 넘어갔을때 자동으로 다시 안으로 넣어줄지 여부
preload boolean false URL을 입력하여 다른 페이지를 팝업으로 띄울때 팝업을 초기화 하면서 해당페이지를 로딩할것인지 처음 열때 로딩할것인지 여부
onOk Function null 확인버튼을 눌렀을때 발생되는 이벤트
onCancel Function null 취소버튼을 눌렀을때 발생되는 이벤트
onOpen String null

팝업이 열릴 때마다 발생되는 이벤트

지정한 명칭의 이벤트 핸들러 메서드는 불러오는 팝업 페이지의 Controller 에 구현되어 있어야 합니다.

지정한 메서드의 첫번째 인자로 onOpenData 가 들어옵니다.

url을 지정하여 다른 페이지를 가져왔을때만 사용가능 합니다.
onOpenData Object null

onOpen 이벤트의 첫번째 인자로 지정되어 팝업에 전달될 데이터

url을 지정하여 다른 페이지를 가져왔을때만 사용가능 합니다.

open 함수의 첫번째 argument 로 넣어줘도 됩니다(동적인 데이터 전달 시 사용).
onClose Function null

팝업이 닫힐 때마다 발생되는 이벤트.

이벤트 콜백의 메서드는 인자로 onCloseData 가 들어옵니다.

팝업 페이지에서 자신을 닫을때는 Controller에서 다음과 같이 처리 합니다.

this.caller.close(onCloseData)

url을 지정하여 다른 페이지를 가져왔을때만 사용가능 합니다.
onCloseData Object null

onClose 이벤트의 첫번째 인자로 지정되어 팝업의 부모페이지에 전달될 데이터

url을 지정하여 다른 페이지를 가져왔을때만 사용가능 합니다.

close 함수의 첫번째 argument 로 넣어줘도 됩니다(동적인 데이터 전달 시 사용).

예제

Examples

1. 특정영역(.popupArea)을 팝업으로 띄우기


<div title="Popup Example" class="popupArea" style="height: 200px;border: 3px dotted #080;">이 블록이 팝업으로 보이나요?</div>

<script type="text/javascript">
   	N(".popupArea").popup().open();
</script>

2. 다른 (블록)페이지를 팝업으로 띄우기


<script type="text/javascript">
   	N().popup("html/indx/header.html").open();
</script>

3. 다른 (블록)페이지를 드래그 가능한 팝업으로 띄우기


<script type="text/javascript">
   	N().popup({
   		url: "html/indx/header.html",
   		title: "Global Navigation Bar",
   		width: 800,
   		draggable: true
   	}).open();
</script>

4. 팝업에 데이터 전달하고 팝업으로부터 데이터 받기

4.1 팝업페이지(popup.html)


<!-- Style -->
<style>
	#__POPUP01 #result { width: 400px; height: 300px; border: 1px solid #080; }
</style>

<!-- View -->
<article id="__POPUP01" title="onOpen Example">
	<p id="result"></p>
	<button id="close">닫기</button>
</article>

<!-- Controller -->
<script type="text/javascript">
N("#__POPUP01").cont({
    init : function(view, request) {
    	var caller = this.caller; // caller는 현제 팝업페이지를 열어준 N.popup 인스턴스
    	N("#close", this.view).click(function() { // 닫기버튼에 이벤트 바인드
    		caller.close("Hello~ parent's Opener"); // close 함수의 첫번째 인자로 onOpenData 데이터 전달
    	});
    },
    // 팝업이 열릴때마다 실행될 함수
    onOpenFunc : function(onOpenData) {
    	N("#result", this.view).text(onOpenData);
    }
});
</script>

4.2 부모페이지(parent.html)


<!-- Style -->
<style>
	#__POPUP01 #result { width: 400px; height: 300px; border: 1px solid #080; }
</style>

<!-- View -->
<article id="__POPUP02" title="onOpen Example">
	<p id="result"></p>
</article>

<!-- Controller -->
<script type="text/javascript">
N("#__POPUP02").cont({
    init : function(view, request) {
	   	var popup = N().popup({
	   		url: "./popup.html",
	   		onOpen: "onOpenFunc", // 팝업이 열릴때마다 실행될 함수명
	   		onClose: function(onCloseData) { // 팝업이 닫힐때마다 실행 될 onClose 이벤트 핸들러
	   			N("#result", this.view).text(onCloseData);
	   		}
	   	});
	   	popup.open("Hello~ popup's onOpen!"); // open 함수의 첫번째 인자로 onOpenData 데이터 전달
    }
});
</script>
