Context(N.context)

개요

Context(이하 N.context)는 Natural-JS 의 생명주기(Life-Cycle, 페이지가 표시되고 다른 URL로 redirect 되기 전까지) 안 에서의 데이터 저장소 역할을 수행 합니다. Natural-JS 의 환경설정 값(Config)과 메시지등이 N.context 객체에 저장 됩니다.

함수(Methods)

Name Arguments Type Return Description
attr - - N.comm
name String - 저장할 값의 key를 입력 합니다.
obj Object - 저장할 값에 해당하는 object를 지정 합니다. obj 인자를 지정하지 않으면 값을 가져오고 지정하면 해당 값으로 저장합니다.

예제

Examples

1. Context 에 값 정의


<script type="text/javascript">
	N.context.attr("globalInfo", {
		userId : "jeff1942",
		userNm : "Jeff beck"
	});
</script>

2. Context 에 정의된 속성값 가져오기


<script type="text/javascript">
	var globalInfo = N.context.attr("globalInfo");
</script>
