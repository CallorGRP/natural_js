Button(N.button)

개요

Button(이하 N.button) 은 a, input button, button 과 같은 버튼 요소들을 공통된 모양과 공통된 동작을 가진 버튼을 만들어 주는 컴포넌트입니다.

API DEMO

생성자

Constructor

Name Arguments Type Return Description
N.button - - N.button N.button의 객체 인스턴스를 생성합니다.
context window or jQuery Object - 기본옵션의 context 옵션과 같습니다.
opts JSON Object - 기본 옵션을 JSON Object 형태로 지정 합니다.
N(obj).button - - N.button

N.button의 객체 인스턴스를 jQuery 플러그인 방식으로 생성합니다.

N.button과 동작되는 로직은 같으며 내부적으로 N()의 첫번째 인자가 N.button()의 첫 번째 인자로 지정 됩니다.
opts JSON Object - N.button의 opts 옵션과 같습니다.

기본옵션

Default options

Name Type Default Required Description
context jQuery Object obj ○ 버튼컴포넌트화 할 버튼 요소를 지정합니다.
size String "medium"

크기

smaller, small, medium, large, big
color String "white"

색깔

white, blue, skyblue, gray
disable boolean false 초기화시 비활성 여부
effect boolean true 활성, 비활성 시 Fade 효과 적용 유무
customStyle boolean false 사용자 정의 스타일 적용 유무

값이 true 이면
css/natural.ui.css 파일에 정의된
a.btn_mouseover__, input[type='button'].btn_mouseover__, button.btn_mouseover__
a.btn_mousedown__, input[type='button'].btn_mousedown__, button.btn_mousedown__
a.btn_mouseup__, input[type='button'].btn_mouseup__, button.btn_mouseup__
a.btn_mouseout__, input[type='button'].btn_mouseout__, button.btn_mouseout__
스타일이 적용 됩니다.

함수

Methods

Name Arguments Type Return Description
context - - jQuery Object 컴포넌트를 적용한 요소를 반환 합니다.
selector String - selector 문자열을 입력하면 해당 context 하위 요소에서 선택된 요소를 반환 합니다.
enable - - N.button 버튼을 활성화 합니다.
disable - - N.button 버턴을 비활성화 합니다.

예제

Examples

1. 선언형 옵션을 적용하여 버튼생성


<a id="aTag" data-opts='{ "color" : "white", size : "big" }'>a</a>
<input id="inputButtonTag" type="button" value="input button" data-opts='{ "color" : "skyblue" }' />
<button id="buttonTag" data-opts='{ "color" : "gray", size : "large" }'>button</button>

<script type="text/javascript">
   	N("#aTag, #inputButtonTag, #buttonTag").button();
</script>

2. 명령형 옵션을 적용하여 버튼생성


<a id="aTag">a</a>

<script type="text/javascript">
   	N("#aTag").button({
   		color: "blue",
   		size: "small"
   	});
</script>
